Execute os comandos no terminal:
pip install django
pip install djangorestframework

Depois de instalada as bibliotecas, execute o comando no terminal:
source venv/bin/activate

Esse comando ativa o ambiente virtual do python na máquina
agora vc só precisa rodar o sistema usando o comando:
python manage.py runserver

Para abrir o sistema use o link:
127.0.0.1:8000/admin
usuario: eliakim
senha: 123456

ou o link:
127.0.0.1:8000/
que mostrará a url da API para vc manipular com seu html
